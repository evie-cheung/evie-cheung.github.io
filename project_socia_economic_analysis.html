Analysis Report on Income Inequality, Unemployment, and Social Issues

Project Type: Data Analysis Project

Description: Examined the correlations between income inequality, unemployment, education rates, and social issues (suicides and crime rates) in New Zealand using historical data and Python visualisations.

Technologies Used: Python, Jupyter Notebook

Outcome: Discovered that unemployment is moderately correlated with social instability (suicides and crime), while higher tertiary education rates are linked to lower unemployment. Income inequality showed a weak negative correlation with social issues.
